AppleScript with ControlPlaneX

This page gives examples how to use AppleScript with ControlPlaneX.

The following script will get the current context from ControlPlaneX:

  1. Use Script Editor to create an AppleScript such as: 
                                tell application "ControlPlaneX"
                                get current context
                            end tell

To force ControlPlaneX to change to a different context using the following script:

  1. Use Script Editor to create an AppleScript with the following: 
                                tell application "ControlPlaneX"
                                set current context to "<Context Name>"
                            end tell
    Where <Context Name> is the name of a context you have created.

The following script will report if the "Sticky" option is set in ControlPlaneX:

  1. Use Script Editor to create an AppleScript with the following: 
                                tell application "ControlPlaneX"
                                get sticky
                            end tell

The following script will set the "Sticky" option:

  1. Use Script Editor to create an AppleScript with the following: 
                                tell application "Controls
                                set sticky to true
                            end tell
